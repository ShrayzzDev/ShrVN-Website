Add image to character

In ShrVN, each character have images associated with it.
It will search images in the project_root/images/In-Game_name/ folder.
To add an image to a character you use that line: 

    
      From character_instance Add image_path As image_name

From :

  • It's the keyword used to select a character and do something with it. In the character file, you can only add an image.
    Keep in mind that you can't use From statements meant for the main script files here.

character_instance :

  • Instance name you used when creating the character.

Add :

  • Key word used to specify what From statement you use.

image_path :

  • Path to the image from the /images/In-Game_name folder
    If the file is directly in it, just put the file name. Don't forget to put the image name in the path.
    You also need to put the extension of the file in the path.
    NOTE : Only .jpg, .jpeg and .png files are supported for the moment.

As :

  • Keyword used to link to the image_name

image_name :

  • Name used to identify the image later in the script.
    Two images in the same character can't have the same name, but two images from different characters can.
