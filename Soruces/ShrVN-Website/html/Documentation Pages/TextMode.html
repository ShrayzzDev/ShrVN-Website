Text Mode

This defines how the game is going to show text at the start of a new game. You have two choices :

  • ADV

    This one is the most common one in visual novels. In this mode, text is shown in a block very often at the bottom of the screen.
    The engine lets you chose your own coordinates for this block, see coordinates for more infos on that
    This is the default mode.

  • NVL

    This one is found less often but this is still used by some very popular VNs. Text can take the entire screen, while images can still be shown in the background.
    NOTE : Most parameters in the InGameOverlay.shvrn files are unused while in this mode.


NOTE : You can switch these modes in the main script, they are not definitive as this only set the one used at the beginning. (you can also never change them, that works too)
For more infos on that, see the page about the change text mode statement.
