In Game Option Position

This parameter tells the engine where to show options in the overlay.

  • outside_right

    Tells to put them at the right, outside the main text block. In this disposition, each element stack each other
    This is the default value.

  • outside_left

    Tells to put them at the left, outside the main text block. Like the right one, each element stack each other

  • inside_bottom

    Tells to put them at the left, outside the main text block. In this disposition, each element are side by side

  • none

    Don't show them at all, they still are accessible via the menu by hitting escape.

NOTE : These have importance only in the ADV text mode, in NVL, this parameter is always in none.
