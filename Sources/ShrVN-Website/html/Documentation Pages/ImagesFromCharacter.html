Show an image from a character

To show an image from a character, you add a line like this one : 

		
			From character_instance Show image_name At x_coord,y_coord

With that, it will show the given image from the given character at given coordinates.
But you can also do it like that : 

		
			From character_instance Show image_name With movement_name

This will create a character and apply given movement. More infos on movement here
