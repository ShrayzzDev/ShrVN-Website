Welcome to the ShrVN Website !

Here you will find everything you'll need about ShrVN.
ShrVN is a game engine written in C++ designed to make Visual Novels using its own scripting language.

In the "Documentation" tab you'll find everything you need about the scripting language and how the engine creates games.
You'll also find a "Get Started" segment in this tab. This will go over the basis on how to get your own game to run.

In the download you'll find every information needed on how to run the engine.

In the "about" tab you'll find information about the gigantic team making this, composed by . . . myself only.
